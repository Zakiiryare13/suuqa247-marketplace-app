# Suuqa24/7 – Marketplace & Marketing System

Production-ready digital system built for Somali businesses, combining marketplace listings, marketing services, and real-time admin control.

## Overview
Suuqa24/7 is a cross-platform system designed to operate reliably in low-connectivity environments.  
It integrates a mobile application, backend services, and admin workflows into a single scalable solution.

The system supports multiple business use cases including e-commerce, real estate listings, hotel services, and professional marketing requests.

## Key Features
- Marketplace system (products, rentals, hotel listings)
- Real-time admin panel (add, edit, disable content)
- Digital & offline marketing request workflows
- WhatsApp automation with pre-filled messages
- Live updates using Firebase Realtime Database
- Offline-safe UX for weak or unstable networks
- Multi-language support (Somali & English)

## System Architecture
- **Frontend:** Flutter (Android & iOS)
- **Backend:** Firebase Realtime Database
- **Integrations:** WhatsApp API
- **UI:** Material 3
- **Design Focus:** Performance, reliability, and usability

## Role & Responsibilities
- System architecture and technical design
- Flutter application development
- Admin panel and backend integration
- WhatsApp automation flows
- Offline-first UX implementation
- Production-ready MVP delivery
